Basketball Clicker - это инкрементальная игра в стиле кликера с простой 2D графикой. Игрок кликает по баскетбольному мячу, чтобы зарабатывать очки, которые можно тратить на покупку виртуальных игроков.

Реализация:
Система коллекции игроков: Реализована на основе ScriptableObjects с настраиваемыми параметрами для каждого игрока (имя, редкость, стоимость, базовый доход).
Инкрементальная экономика: Балансировка доходов и стоимостей по нелинейной прогрессии для поддержания игрового темпа.
Система авто-кликинга: Автоматизированный доход от купленных игроков с использованием Coroutines для плавного прироста очков.
Система сохранений: Локальное сохранение прогресса игрока (баланс, купленные игроки) с использованием PlayerPrefs и JSON-сериализации.
Оптимизация UI: Динамическое обновление интерфейса через события (C# events) для минимизации постоянных проверок.
Система улучшений: Масштабируемая архитектура для добавления бустов и временных бонусов.

Технологии:
Язык: C#
Движок: Unity 2022.3.x
UI Система: Unity UI (Canvas, Buttons, TextMeshPro)
Система контроля версий: Git
Доп. библиотеки: Newtonsoft.Json (для работы с JSON)

Установка и запуск:
Клонируй репозиторий: git clone https://github.com/ТВОЙ_ЮЗЕР/basketball-clicker.git
Открой проект в Unity версии 2022.3.x LTS или выше.
Открой сцену Assets/Scenes/MainGame.unity.
Нажми "Play" в редакторе Unity.
